# ISIMIP3BASD (Vendored)

This directory contains a vendored copy of the ISIMIP3BASD code for bias adjustment and statistical downscaling.

**Original Source:** https://github.com/ISI-MIP/isimip3basd  
**Reference:** Lange, S. (2019). Trend-preserving bias adjustment and statistical downscaling with ISIMIP3BASD (v1.0). Geoscientific Model Development, 12(7), 3055-3070. https://doi.org/10.5194/gmd-12-3055-2019

**Copyright:** (C) 2019 Potsdam Institute for Climate Impact Research (PIK)

## Note

To use this code, place the following files from the original repository in this directory:
- `bias_adjustment.py`
- `statistical_downscaling.py`
- `utility_functions.py`

These files are not included in this repository to comply with licensing requirements.
Download them from: https://github.com/ISI-MIP/isimip3basd
