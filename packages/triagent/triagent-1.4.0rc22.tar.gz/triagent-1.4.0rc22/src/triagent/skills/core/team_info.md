## Quick Lookups

### Find Team by Service

To find which team owns a service, search for the `CloudRoleName` in the Service to Team Mapping table above.

### Find Team by Module

To find which team owns a module or feature area:
1. Search the Domain Ownership table above by module name or alias
2. Or check the `domain-mapping.json` file for programmatic lookups

### Find Area Path by Team

Use the format:
```
Audit Cortex 2\Omnia Data\[Parent]\[POD]\[Team Name]
```

Where `[Parent]` is either:
- `Omnia Data Management` (for Data Acquisition and Preparation, Data Management and Activation)
- `Omnia Data Automation` (for Data In Use, Omnia JE)
