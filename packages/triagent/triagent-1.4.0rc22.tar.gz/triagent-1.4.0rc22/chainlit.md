# Welcome to Triagent

Triagent is a Claude-powered assistant for Azure DevOps automation.

## What I can help with:

- **Work Items**: Create, update, search, and manage work items
- **Pull Requests**: Review, comment, and manage PRs
- **Code Search**: Find code patterns and search repositories
- **Pipelines**: View runs, logs, and trigger pipelines
- **Wiki**: Search and read documentation

## Getting Started

1. Select a team profile from the dropdown
2. Wait for the session to initialize
3. If prompted, complete Azure CLI authentication
4. Start chatting!
