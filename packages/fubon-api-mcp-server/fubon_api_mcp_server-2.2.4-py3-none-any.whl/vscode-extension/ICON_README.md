# Icon Placeholder

請將 icon.png 放置在此目錄下。

## 圖示需求

- **尺寸**: 128x128 pixels
- **格式**: PNG
- **背景**: 透明
- **內容建議**: 
  - 富邦證券 logo (需取得授權)
  - 或使用簡單的圖表/金融相關圖示
  - MCP 字樣

## 設計建議

1. 使用富邦品牌色彩 (藍色/綠色)
2. 保持簡潔清晰
3. 在小尺寸下仍然清晰可辨
4. 遵循 VS Code Extension 圖示設計規範

## 臨時方案

在沒有正式 icon 之前，可以：
1. 創建一個簡單的文字 icon (F 字母)
2. 使用線上工具: https://www.figma.com/
3. 或使用 icon 生成器: https://icon.kitchen/
