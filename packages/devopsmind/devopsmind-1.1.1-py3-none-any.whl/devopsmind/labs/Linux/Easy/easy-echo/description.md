# Simple Echo Script

Objective:

Introduce basic Bash scripting by creating a script that prints
a fixed message to standard output.

This lab focuses on correct script creation and exact output,
which are foundational skills in Linux automation.

---
Task Requirements:

Create a Bash script named `echo_hello.sh` in the working directory.

The script must:

- Print exactly:
  Hello DevOpsMind
- Produce no extra output
- Include a single trailing newline (acceptable)

---
Constraints:

- Do NOT print additional text
- Do NOT read input
- Do NOT use advanced shell features
- Validation is local and file-based only
