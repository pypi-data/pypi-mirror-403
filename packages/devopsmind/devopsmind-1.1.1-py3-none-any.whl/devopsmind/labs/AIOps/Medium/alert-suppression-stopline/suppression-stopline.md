# Alert Suppression Stop-Lines

Define explicit conditions where AI-based
alert suppression must immediately disengage.

Use clear, enforceable language.

# Mandatory Re-Exposure Conditions

Define situations where suppressed alerts
must be fully visible again to operators.

Avoid vague language such as:
- "if confidence is low"
- "if the system is unsure"

Write in your own words.
Do not describe tools or implementation details.
