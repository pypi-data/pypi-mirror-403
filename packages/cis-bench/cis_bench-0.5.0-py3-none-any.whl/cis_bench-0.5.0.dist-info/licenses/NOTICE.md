## Third-Party Content

### CIS Security Benchmarks

This software downloads and processes CIS Security Benchmarks from the Center for
Internet Security, Inc. ("CIS") WorkBench platform.

**CIS Benchmark Content is NOT included in this repository.**

This tool:

- Requires users to have their own CIS WorkBench account
- Accesses CIS Benchmarks through user's authenticated session
- Downloads benchmarks the user is authorized to access
- Transforms benchmarks for the user's own compliance workflows
- Does not redistribute CIS content to unauthorized parties
