# Naver Blog SEO Optimizer Skill

## Skill: /naver-blog

네이버 블로그 포스트를 SEO 최적화하여 작성하는 스킬입니다.

### 사용법
```
/naver-blog [주제 또는 키워드]
/naver-blog "맥북 프로 M3 리뷰" --length long --tone professional --type review
```

---

## 실행 워크플로우

이 스킬을 실행할 때 아래 단계를 **순서대로** 따르세요:

### Step 1: 옵션 파싱

사용자 입력에서 옵션을 추출하세요:

| 옵션 | 기본값 | 값 |
|-----|-------|---|
| `--length` | medium | short(1,200자) / medium(2,000자) / long(3,000자) |
| `--tone` | casual | formal / casual / professional |
| `--type` | info | info / review / howto / list |
| `--images` | 3 | 권장 이미지 개수 |

> ⚠️ **최소 1,200자 필수** - 체류시간 확보를 위해 short 옵션도 1,200자 이상

### Step 2: 최신 SEO 가이드라인 수집

**WebSearch**를 사용하여 최신 네이버 SEO 트렌드를 검색하세요:
```
query: "네이버 블로그 SEO 최적화 [현재연도]"
```

**WebFetch**를 사용하여 공식 가이드라인을 확인하세요:
- https://searchadvisor.naver.com/guide/seo-basic-intro (네이버 공식 SEO 가이드)

수집한 최신 정보가 아래 기본 가이드라인과 다르면, 최신 정보를 우선 적용하세요.

### Step 3: 키워드 리서치

**WebSearch**를 사용하여 관련 키워드를 조사하세요:
```
query: "[사용자 키워드] 네이버 블로그"
query: "[사용자 키워드] 후기"
query: "[사용자 키워드] 추천"
```

검색 결과에서 추출할 정보:
- 상위 노출 글의 제목 패턴
- 연관 키워드 및 롱테일 키워드
- 검색 의도 (정보성/상업성/탐색성)

### Step 4: 콘텐츠 작성

아래 **SEO 가이드라인**을 적용하여 블로그 포스트를 작성하세요.

### Step 5: 검증 및 출력

작성된 콘텐츠가 **체크리스트**를 충족하는지 확인 후 최종 출력하세요.

## 네이버 SEO 가이드라인 (기본 참조)

> 이 섹션은 기본 참조용입니다. Step 2에서 수집한 최신 정보를 우선 적용하세요.

### 네이버 검색 알고리즘

#### C-Rank (Creator Rank)
블로그/출처의 **신뢰도**를 평가:

| 요소 | 설명 |
|-----|-----|
| **Context** | 주제별 관심사 집중도 (31개 카테고리) |
| **Content** | 생산되는 정보의 품질 |
| **Chain** | 콘텐츠의 소비/생산 연쇄반응 |
| **Creator** | 블로그의 신뢰도와 인기도 |

#### D.I.A+ (Deep Intent Analysis)
개별 **문서의 품질**과 **검색 의도 매칭**을 평가:

1. **경험 정보** - 실제 사용 후기, 체험 **(가장 중요)**
2. **문서의 주제 적합도** - 검색어와의 관련성
3. **정보의 충실성** - 내용의 완성도
4. **문서의 의도** - 광고성 vs 정보성 판별
5. **독창성** - 표절/복사 여부
6. **적시성** - 최신 정보 여부

## 블로그 지수 최적화 (실전 팁)

### 체류시간 & 스크롤 유도
- **최소 글자 수: 1,200자 ~ 2,000자** (너무 짧으면 체류시간 부족)
- 본문 중간에 **질문 배치**하여 스크롤 유도
- 궁금증을 유발하는 구조로 배치 (답은 아래에)

### 댓글 & 이웃 확보
- 댓글은 블로그 지수에 **매우 중요**
- 글 마지막에 **댓글 유도 문구** 포함
- **이웃 추가 요청** CTA 배치
- 예시: "여러분의 경험도 댓글로 공유해주세요!", "도움이 되셨다면 이웃 추가 부탁드려요 🙏"

### 외부 유입 확대
- 발행 후 외부 플랫폼에 공유 권장:
  - 카카오톡/카카오스토리
  - 링크드인
  - 브런치
  - 밴드
- 외부에서 유입되는 트래픽이 지수에 긍정적

### 롱테일 키워드 전략
경쟁이 낮은 세부 키워드가 효과적:

| ❌ 피해야 할 키워드 | ✅ 효과적인 키워드 |
|------------------|------------------|
| 강남 맛집 | 강남역 3번출구 조용한 파스타집 |
| 맥북 리뷰 | 맥북 프로 M3 영상편집 실사용 후기 |
| 다이어트 | 직장인 점심시간 10분 스트레칭 다이어트 |

## 콘텐츠 작성 가이드라인

### 제목 작성 규칙
- **핵심 키워드를 제목 앞부분에 배치** (필수)
- **40자 이내** (네이버 공식 권장)
- 제목과 내용 일치 (낚시성 = 저품질 판정)
- 클릭 유도: 숫자, 질문형, 리스트형 활용

제목 템플릿 (--type별):
```
[info]   [키워드] 완벽 가이드 | [연도] 최신
[review] [키워드] 솔직 후기 - [기간] 사용 리뷰
[howto]  [키워드] 방법 [N]가지 (초보자 필독)
[list]   [키워드] 추천 TOP [N] - 비교 총정리
```

### 본문 구조

```
[도입부 - 150자]
- 메인 키워드 첫 문단에 필수 포함
- 독자의 문제/니즈 공감
- 이 글에서 얻을 수 있는 가치 제시

[H2: 소제목 1]
- 키워드 포함
- 3-4줄 문단
- 경험 기반 내용 (D.I.A+ 핵심)

[H2: 소제목 2]
...

--- 🎯 스크롤 유도 질문 (본문 중간에 배치) ---
예: "그렇다면 실제로 사용해본 결과는 어땠을까요?"
예: "여기서 가장 중요한 포인트가 나옵니다."
(독자가 계속 스크롤하도록 궁금증 유발)

[H2: 소제목 3]
...

[결론 - 150자]
- 핵심 내용 요약
- 키워드 마지막 언급

--- 💬 댓글/이웃 유도 CTA (필수) ---
예: "여러분의 경험도 댓글로 공유해주세요!"
예: "도움이 되셨다면 이웃 추가 부탁드려요 🙏"
예: "궁금한 점 댓글로 남겨주시면 답변 드릴게요!"
```

**필수 요소:**
- 소제목(H2, H3) **3개 이상**
- 문단 길이 **3-4줄 이내** (모바일 가독성)
- 구체적 수치/날짜/장소 포함
- 리스트, 번호 매기기 활용

### 키워드 배치
| 위치 | 규칙 |
|-----|-----|
| 제목 | 메인 키워드 앞부분 |
| 첫 문단 | 메인 키워드 필수 |
| 소제목 | 키워드 또는 관련어 |
| 본문 | 3-5회 자연스럽게 분산 |
| 마지막 문단 | 키워드 재언급 |

**키워드 밀도: 2-3%** (과도한 반복 = 스팸 판정)

### 이미지 가이드
- **최소 3개** (--images 옵션으로 조절)
- 첫 이미지는 본문 시작 전 또는 첫 문단 후
- 파일명 예시: `macbook-pro-m3-review.jpg`
- Alt 텍스트 필수 설정
- 용량 300KB 이하 권장
- **인포그래픽/차트**: 높은 평가 + 공유 가능성 증가

### 링크 구성
- **내부 링크 1-3개**: 본인의 관련 글
- **외부 링크 1-2개**: 공식 사이트, 권위 있는 출처

## 주의사항 (작성 시 반드시 준수)

### 저품질 판정 요인 - 절대 금지

| 항목 | 기준 | 결과 |
|-----|-----|-----|
| **복붙/표절** | AI 생성물, 타 블로그 복사 | 즉시 저품질 (네이버 AI 탐지 정확도 높음) |
| **키워드 과다 반복** | 제목 3회↑, 본문 10회↑ | 지수 하락 |
| **하루 다량 발행** | 5개 이상/일 | 스팸 판정, 지수 치명타 |
| **발행 후 다수 수정** | 여러 번 수정 | 지수 하락 |
| **낚시성 제목** | 제목과 내용 불일치 | 저품질 판정 |

### 고품질 콘텐츠 핵심
- ✅ 실제 경험 기반 (D.I.A+ 최우선 평가 요소)
- ✅ 구체적 정보 (수치, 날짜, 장소)
- ✅ 독자에게 실질적 도움
- ✅ 자연스러운 문체 (AI 티 나면 안됨)
- ✅ **발행 전 맞춤법 검사 필수** (수정 최소화)

### 네이버 SERP 특성
- 유기적 검색 결과 **4개만 노출** (구글 10개와 다름)
- 경쟁이 치열하므로 품질이 더 중요
