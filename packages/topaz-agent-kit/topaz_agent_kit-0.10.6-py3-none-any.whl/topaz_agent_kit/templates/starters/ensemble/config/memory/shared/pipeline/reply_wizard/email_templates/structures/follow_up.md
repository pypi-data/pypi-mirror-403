# Follow-Up Email Structure

## Template Structure

1. **Greeting** (professional or friendly)
   - Reference previous interaction
   - Show continuity

2. **Context**
   - Brief reminder of previous conversation
   - Current status update

3. **Update/Information**
   - What's new
   - Progress made
   - Any changes

4. **Next Steps**
   - What's needed from recipient (if any)
   - What will happen next
   - Timeline

5. **Closing** (professional or friendly)
   - Offer to answer questions
   - Thank them

## Example Flow

```
[Greeting]
Hi [Name],

Following up on our conversation about [topic] from [date/time].

[Context]
As discussed, we were [previous action/status].

[Update]
I wanted to update you that [new information/progress]:
- Update point 1
- Update point 2

[Next Steps]
[What's needed or what will happen next]

[Closing]
Please let me know if you have any questions.

Best regards,
```
