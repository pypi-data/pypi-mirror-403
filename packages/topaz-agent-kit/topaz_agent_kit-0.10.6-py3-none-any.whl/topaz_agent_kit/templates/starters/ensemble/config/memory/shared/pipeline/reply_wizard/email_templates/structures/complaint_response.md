# Complaint Response Structure

## Template Structure

1. **Greeting** (formal or professional)
   - Acknowledge the complaint
   - Express genuine concern

2. **Apology & Acknowledgment**
   - Sincere apology
   - Acknowledge the specific issue
   - Validate their feelings/experience

3. **Explanation** (if appropriate)
   - Brief, honest explanation
   - Avoid excuses
   - Focus on what happened, not blame

4. **Resolution**
   - Specific steps being taken
   - What will be done to fix the issue
   - Timeline for resolution

5. **Prevention**
   - How this will be prevented in future
   - Process improvements (if applicable)

6. **Closing** (formal or professional)
   - Reiterate apology
   - Offer further assistance
   - Thank them for feedback

## Example Flow

```
[Greeting]
Dear [Name],

Thank you for bringing this matter to our attention. We sincerely apologize for the experience you've had.

[Acknowledgment]
We understand how frustrating [specific issue] must have been, and we take full responsibility.

[Explanation]
[Brief, honest explanation if appropriate]

[Resolution]
We are taking the following steps to address this:
1. [Action 1]
2. [Action 2]
3. [Action 3]

We expect to have this resolved by [timeline].

[Prevention]
To prevent this from happening again, we are [improvement measures].

[Closing]
Again, we sincerely apologize for this experience. If you have any further concerns, please contact us directly.

Best regards,
```
