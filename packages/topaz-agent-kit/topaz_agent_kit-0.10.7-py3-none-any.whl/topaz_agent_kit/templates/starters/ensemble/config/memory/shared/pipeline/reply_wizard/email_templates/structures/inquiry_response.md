# Inquiry Response Structure

## Template Structure

1. **Greeting** (from greetings/)
   - Acknowledge the inquiry
   - Show appreciation

2. **Acknowledgment**
   - Confirm understanding of the question/request
   - Set context if needed

3. **Main Content**
   - Provide clear, concise answer
   - Include relevant details
   - Use bullet points for multiple items
   - Add examples if helpful

4. **Next Steps** (if applicable)
   - What happens next
   - Timeline expectations
   - Required actions from recipient

5. **Closing** (from closings/)
   - Professional sign-off
   - Offer additional help

## Example Flow

```
[Greeting]
Thank you for your inquiry about [topic].

[Acknowledgment]
I understand you're looking for information about [specific aspect].

[Main Content]
Here's what I can share:
- Point 1
- Point 2
- Point 3

[Next Steps]
If you need further assistance, please don't hesitate to reach out.

[Closing]
```
