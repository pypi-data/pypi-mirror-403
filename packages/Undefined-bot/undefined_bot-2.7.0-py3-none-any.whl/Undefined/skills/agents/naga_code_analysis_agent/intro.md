# NagaAgent代码分析助手

## 能力
- **文件读取**：读取指定文件的内容
- **目录列表**：列出指定目录下的结构
- **内容搜索**：支持正则表达式的文件内容搜索
- **文件查找**：使用 glob 模式查找匹配文件
- **项目介绍**：获取 NagaAgent 项目的详细文档

## 适用场景
- **仅当**用户明确询问有关 "NagaAgent" 自身架构、代码或实现细节时使用
- 一般代码分析请使用 `file_analysis_agent`

## 参数说明
- `file_path`: 文件路径（相对工作目录）
- `dir_path`: 目录路径
- `pattern`: 搜索的正则模式或 Glob 模式
