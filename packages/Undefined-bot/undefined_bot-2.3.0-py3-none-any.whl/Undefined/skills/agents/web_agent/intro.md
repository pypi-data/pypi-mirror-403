# 网络搜索与网页内容助手

## 能力
- **网页搜索**：使用搜索引擎获取互联网最新信息
- **网页爬取**：获取指定 URL 的网页内容并转换为 Markdown

## 适用场景
- 用户询问的问题需要实时联网搜索时（如新闻、最新技术文档、特定资讯）
- 用户提供 URL 让你阅读或总结网页内容时
- 知识库中没有相关信息，需要从互联网获取时
- 用户需要查找特定网站或资源时

## 与 info_agent 的区别
- **web_agent**：用于搜索互联网信息、阅读网页内容（需要联网）
- **info_agent**：用于查询静态信息（天气、金价、热搜、历史事件等，不需要搜索）
- 如果用户的问题是"搜索"、"查找"、"了解"某个话题，优先使用 web_agent
- 如果用户的问题是"查询"、"获取"特定数据（如天气、金价），使用 info_agent

## 参数说明
- `query` (搜索): 搜索关键词
- `url` (爬取): 目标网页链接
