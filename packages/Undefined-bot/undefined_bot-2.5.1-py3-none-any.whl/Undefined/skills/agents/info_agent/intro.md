# 静态信息查询助手

## 能力
- **生活服务**：查询指定城市天气、查询实时黄金价格
- **热搜榜单**：获取百度、微博、抖音热搜榜，获取腾讯新闻
- **文化历史**：查询历史上的今天
- **网络工具**：网络诊断(TCP Ping/网速测试/连通性检查)、Whois域名查询、Base64编解码、Hash计算
- **其他查询**：查询QQ等级

## 适用场景
- 用户查询天气、金价时
- 用户想看新闻或热搜时
- 用户查询历史事件时
- 用户需要测试网络连通性、查询域名信息时
- 用户需要进行 Base64 编解码或计算 Hash 时
- 用户查询 QQ 等级时

## 不适用场景
- **不适用于网络搜索**：如果用户需要搜索互联网上的信息（如最新技术文档、特定资讯），请使用 web_agent
- **不适用于网页内容获取**：如果用户提供 URL 需要阅读或总结网页内容，请使用 web_agent

## 参数说明
- `city` (天气): 城市名称（如：北京）
- `qq` (QQ等级): 目标 QQ 号码
- `target`/`host` (网络工具): 目标域名或 IP
- `text` (编码/哈希): 需要处理的文本内容
