# AI URL to Markdown

Extract article content from URL and convert to clean Markdown.

## Usage
```bash
praison run ai-url-to-markdown https://example.com/article
praison run ai-url-to-markdown https://example.com/article --no-images
```

## Output
- `article.md` - Clean markdown content
- `images/` - Downloaded images
- `metadata.json` - Title, author, date
