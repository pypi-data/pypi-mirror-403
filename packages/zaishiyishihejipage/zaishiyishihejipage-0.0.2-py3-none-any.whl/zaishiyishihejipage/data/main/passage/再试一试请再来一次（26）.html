再试一试请再来一次（26）

周三，烛明再次前往西北荒凉之地去找仲夏。

还有十天就要考试了，仲夏可谓十分焦虑，虽然其自述最近一直在看综艺打游戏，好像满不在乎似的，但烛明很清楚，焦虑的人都有一个共性——嘴硬。

做了一个小时的公交车，烛明成功到达了仲夏所在的学习机构。得知烛明已经到来，仲夏飞快的跑下楼。

“这个给你。”烛明说，然后递给仲夏一个中心嵌有一颗玻璃珠厚度和直径正好放在手心里的不锈钢块儿。

“明，这是什么？”仲夏问。

“上回说给你带的法器。”

“啊！他怎么用？”

“拿在手里。”仲夏照做，“瞄准目标。”仲夏对准前面的灌木，“半蹲对抗后坐力然后念刻在背面的咒语。”轰的一声，灌木被夷平的渣都不剩。

“/*仲夏粗话*/，这么猛的吗？”

“对的，混凝土也能轰开。”

接下来两个人去到了弘阳广场，仲夏骑车拉着烛明。在弘阳广场，仲夏带烛明去到了一家小火锅店，这是整个广场规模最大而且食物最全的火锅店，在这里两个人边吃边聊，一会儿聊到仲夏的人生规划，一会儿又聊到烛明杀穿佛界，再回到仲夏近来的情况，可以说相当一段时间以来，仲夏都没有如此轻松过了。

然后仲夏又带烛明去台球店打球。烛明不会打台球，但相比于自己进球，更兴奋的是仲夏进球。打了四局，时间到了，两个人坐在球场里面歇了一会儿，隐约听到对面印象济南商圈有人投毒。

“他是不是说投毒了？”烛明问仲夏。

“他说的好像是……毒杀宠物狗？”仲夏说。

“既然这样……去瞅瞅？”

“走！”

于是二人过马路来到了印象济南。

此时天已经要黑了，一般来说如果是给猫猫狗狗的投毒的话，应该是把毒药放在食物里面然后碰到路边，但这黑灯瞎火的找那丁点儿的烤肠状物确实是有点儿困难的。

这是仲夏突发奇想，狗狗可比烤肠大多了，完全可以跟着一只狗狗去找毒物，接着一只带着项圈的狗狗就进入了他们的视野。很快，他们顺利的找到了一组毒物，混在烤肠里面。

狗狗吃了下去，不出一分钟，就抽搐了起来。烛明果断上前，抓住狗狗的后腿把它倒置，五指猛抓它的胃，硬生生把刚吃进去的毒药给挤了出来。劫后余生的狗狗虚弱的趴在地上，一个声音从烛明和仲夏身后传来：“喜欢多管闲事？”回头一看，大概五六个人一团的逼近他们。

“还真是。”说罢，仲夏一甩手，把他们全部轰飞，有的轰到绿植里面，有的直接躺倒在了地上。

“右手边还有一个，墙里面，毛坯房，非承重墙。”烛明对仲夏说。

“明白！”然后仲夏对准了那堵混凝土钢筋墙，把他们最后一个同伙也给轰的再起不能。

收拾完这伙人，仲夏转头一看，狗狗已经恢复活力，甚至奔跑着扑向一辆婴儿车。烛明顺着仲夏的目光看去，迅速用法力把狗狗拽了回来，让它的后脖颈飞到了自己的手上。

直到现在，一个老人才走到烛明旁边找自己的宠物。

“这事儿其实也挺复杂的。”仲夏说。

“确实，施暴者养宠不文明让其他人受害，受害者投毒又变成了施暴者。”烛明说。

“明怎么看这个事儿？”

“我看的话……我喜欢老鼠，是因为我是医学生，天天做实验根本不怕老鼠，类似的，只有人们不害怕动物，才能谈论宠物友好的事情。”

“对呀。而且……”，仲夏说，“我现在也有了一些解决问题的能力，‘轰！’，有的事情，也该由我们亲自去解决了。”

今日亦是平和的一天，明天，要继续奋斗了。
