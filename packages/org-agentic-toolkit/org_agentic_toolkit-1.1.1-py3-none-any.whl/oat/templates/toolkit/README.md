# Org Agentic Toolkit

This directory contains the toolkit implementation for the Org Agentic Toolkit (OAT).

## Structure

- `oat`: CLI wrapper script
- `targets.yaml`: IDE/Agent target mappings
- `schemas/`: JSON schemas for validation
- `templates/`: Project initialization templates

## Usage

The toolkit is typically installed as a Python package and accessed via the `oat` command.

See the main README.md for usage instructions.
