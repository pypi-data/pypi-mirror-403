# IT Support FAQ

## VPN Access

### How to reset VPN password?

To reset your VPN password:
1. Go to https://vpn.company.com
2. Click "Forgot Password"
3. Enter your email address
4. Check your email for reset link
5. Follow instructions in email

Contact IT support if issues persist: support@company.com

### How to install VPN client?

Download VPN client from:
- Windows: https://vpn.company.com/download/windows
- macOS: https://vpn.company.com/download/mac
- Linux: https://vpn.company.com/download/linux

Installation steps:
1. Download installer for your OS
2. Run installer with admin rights
3. Enter your credentials
4. Connect to "Main Office" profile

## Email Issues

### Email not syncing on mobile?

Check these settings:
1. Server: mail.company.com
2. Port: 993 (IMAP) or 587 (SMTP)
3. SSL: Enabled
4. Username: your_email@company.com
5. Password: your email password

Try removing and re-adding account if issues persist.

### How to setup email signature?

In Outlook:
1. File → Options → Mail
2. Click "Signatures"
3. Click "New"
4. Enter signature text
5. Set as default for new messages

Signature template:
```
[Your Name]
[Your Title]
Company Name
Email: your_email@company.com
Phone: +7 (XXX) XXX-XX-XX
```
