Lence Component Demos

Visual demos for Lence web components. Each page renders a component with static mock data.

Components

  • lence-table
    Simple table with sorting. Displays query results in tabular format.
    table/simple-table.html
  • lence-chart
    ECharts-based charts. Line, bar, area, pie, and scatter charts.
    chart/echarts-chart.html
  • lence-layout
    App layout with sidebar navigation and content slot.
    layout/layout.html
  • lence-page
    Markdoc page renderer with embedded charts and tables.
    page/page.html

Running

Start Vite dev server and open this page:

npx vite --open /lence/frontend/components/__tests__/
