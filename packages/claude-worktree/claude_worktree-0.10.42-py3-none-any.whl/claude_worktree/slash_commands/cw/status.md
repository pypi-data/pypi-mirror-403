# Show Worktree Status

Display current worktree metadata and all worktrees for the repository.

## Usage

/cw:status

## Output

Shows:
- Current worktree feature branch
- Base branch it was created from
- Path to base repository
- Table of all worktrees with their status

## Example

```bash
/cw:status
```
