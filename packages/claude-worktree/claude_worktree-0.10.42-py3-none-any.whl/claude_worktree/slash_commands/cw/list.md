# List All Worktrees

Show all git worktrees for the current repository with their status.

## Usage

/cw:list

## Output

Displays a table with:
- Branch name
- Status (active, clean, modified)
- Worktree path

## Example

```bash
/cw:list
```

---

Execute the command:

```bash
cw list
```

After execution, show the list of worktrees and highlight any important information (modified worktrees, current worktree, etc.).
