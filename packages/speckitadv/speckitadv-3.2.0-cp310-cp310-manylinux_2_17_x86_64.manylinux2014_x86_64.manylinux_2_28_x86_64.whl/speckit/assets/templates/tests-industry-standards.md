# Testing Standards

## Test Pyramid

| Level | % | Speed |
|-------|---|-------|
| Unit | 60-70% | <100ms |
| Integration | 20-30% | <1s |
| E2E | 5-10% | Slow |

## FIRST Principles

- **F**ast, **I**ndependent, **R**epeatable, **S**elf-validating, **T**imely

## AAA Pattern

`Arrange -> Act -> Assert`

## Edge Cases

Null/empty, boundaries (0,1,max), errors, unicode
