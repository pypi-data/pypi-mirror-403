{# Contenu de la documentation Lucy Assist #}

Guide Lucy

Aide contextuelle

Lucy détecte automatiquement la page où vous vous trouvez pour vous proposer une aide ciblée.

Recherche intelligente

Trouvez rapidement n'importe quel élément en posant simplement la question.

Exemple : "Trouve-moi le client Dupont"

Création assistée

Créez de nouveaux éléments en discutant avec Lucy qui vous guide étape par étape.

Exemple : "Crée un nouveau membre nommé Jean Martin"

Analyse de bugs

Signalez un problème et Lucy analysera pour détecter d'éventuels bugs.

Exemple : "Je n'arrive pas à enregistrer le formulaire, j'ai une erreur"

Raccourcis clavier

  • Echap Fermer Lucy
  • Entrée Envoyer le message
