# Security Policy

We take the security of the Agent CLI SDK seriously and appreciate responsible disclosures.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅
| < 0.1.0 | ❌

We follow semantic versioning. If you rely on a different release line, please reach out.

## Reporting a Vulnerability

If you believe you have found a security issue, please report it via [GitHub Security Advisories](https://github.com/rbbtsn0w/agent-cli-sdk/security/advisories/new) or email the maintainers with the following details:

- A description of the issue and potential impact
- Steps to reproduce or proof of concept
- Affected version(s) and environment details

Please **do not** open public GitHub issues for security reports.

## Response Process

1. We will acknowledge receipt within 3 business days.
2. We will investigate and provide an initial assessment and mitigation plan.
3. Once a fix is available, we will coordinate disclosure and release notes. If needed, we may ask for your help verifying the fix.

Thank you for helping keep the community safe.
