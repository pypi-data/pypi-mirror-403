# Beta Release Disclaimer

This is a **BETA** release of the pyWATS API and Client. Please read this disclaimer carefully before use.

## API Status

### Core API Interface
- **Stability Level:** Relatively Stable
- **Status:** The core API interface is considered relatively stable and is expected to remain largely consistent through the beta period
- **Breaking Changes:** Minor breaking changes are possible but unlikely
- **Subject to Change:** The API specification and behavior remain subject to change based on user feedback and testing

### Client and GUI
- **Stability Level:** Less Stable
- **Testing:** Limited testing and validation
- **Known Issues:** The client and GUI components have not been thoroughly tested or validated in production environments
- **Recommended For:** Development, testing, and evaluation purposes
- **Not Recommended For:** Production deployments or critical workflows at this time

## Beta Responsibilities

As a beta tester, you are expected to:

1. **Report Issues:** Please report any bugs, unexpected behavior, or issues encountered
2. **Provide Feedback:** Share your experience and suggestions for improvement
3. **Test Thoroughly:** Help validate the API and client functionality in your environment
4. **Understand Limitations:** Accept that the beta release may have limitations or incomplete features

## Support and Feedback

- **Issue Reporting:** Report bugs and issues through the project repository
- **Feature Requests:** Submit enhancement requests and suggestions
- **Questions:** Use the documentation and examples for guidance

## Upgrade Path

When upgrading from beta to stable releases:
- Review the [Migration Guide](./release/0.2.0b1/MIGRATION.md) for version-specific changes
- Test thoroughly in a non-production environment first
- Allow time for validation before deploying to production
