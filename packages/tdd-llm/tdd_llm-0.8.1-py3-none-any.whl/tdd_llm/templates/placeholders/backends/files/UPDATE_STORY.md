**Update story:** Edit the epic markdown file directly.

1. Find the epic file: `docs/epics/{epic_id}-*.md`
2. Locate the task section: `## {task_id}: {title}`
3. Edit the title, description, or acceptance criteria
4. Save the file

Task format:
```markdown
## T1: Task title

Task description here.

**Acceptance Criteria:**
- Criterion 1
- Criterion 2
```
