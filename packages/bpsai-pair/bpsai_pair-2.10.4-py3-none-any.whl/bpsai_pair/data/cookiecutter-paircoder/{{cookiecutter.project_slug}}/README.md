## Project Structure

```
.paircoder/           # PairCoder configuration & state
├── config.yaml       # Project settings
├── capabilities.yaml # AI capability manifest
├── context/          # Project context for AI
│   ├── state.md      # Current sprint/task status
│   ├── project.md    # Project overview
│   └── workflow.md   # How we work here
├── plans/            # Sprint plans (.plan.yaml)
└── tasks/            # Task files (.task.md)

## Common Commands

```bash
# Planning
bpsai-pair plan new my-feature --type feature
bpsai-pair plan status

# Tasks
bpsai-pair task list
bpsai-pair task update TASK-001 --status done

# Context
bpsai-pair status
bpsai-pair context-sync --last "Completed X" --next "Working on Y"
```
